# Expo Web

> If you're using Next.js, use the [Next.js docs](/get-started/web/next) instead.

If you're using `expo start:web`, this section is for you.

## Install

```bash
yarn add -D @expo/webpack-config
```

## Configure

Create a custom `webpack.config.js` file:

```js
const createExpoWebpackConfigAsync = require('@expo/webpack-config')

module.exports = async function (env, argv) {
  const config = await createExpoWebpackConfigAsync(
    {
      ...env,
      babel: {
        dangerouslyAddModulePathsToTranspile: ['dripsy', '@dripsy'],
      },
    },
    argv
  )

  return config
}
```
